# Shaxriyor Ismatov Portfolio (`fronted_2`)

Minimalistic, futuristic backend-engineer portfolio built with:
- Vite
- React + TypeScript (strict)
- Tailwind CSS

## Run

```bash
cd fronted_2
npm install
npm run dev
```

## Build

```bash
cd fronted_2
npm run build
npm run preview
```

## Structure

```text
src/
  components/
    ui/
    layout/
    sections/
    icons/
  hooks/
  lib/
  pages/
```
